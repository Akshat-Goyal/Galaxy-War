# Sky Force

The player needs to protect the plane from the enemies and kill boss enemy using missiles within time to win the game.

## HOW TO RUN

1. 6 Keys to move the plane.
    - ArrowUp - Top
    - ArrowDown - Bottom
    - ArrowLeft - Left
    - ArrowRight - Right
    - R - Down
    - F - Up

2. 2 Keys to rotate the plane. 
    - Q - rotate left
    - E - rotate right

3. 1 Key to launch missile.
    - P - launches missile
